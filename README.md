# Store_Backend

## EndPoints
### userEndPoint
#### GET all user
> http://localhost:3000/api/users
#### GET one user
> http://localhost:3000/api/users/:id

#### Post create user
> http://localhost:3000/api/users/

#### Delete delete one user
> http://localhost:3000/api/users/:id
#### Patch  update one user
> http://localhost:3000/api/users/:id
